
## Creation Workflow

### Three Levels of Creation

**Level 1: Material Definition (The "What")**
- Define what the object consists of
- Select base material type
- Specify finish, texture, color, pattern
- Set optical properties (translucency, reflectance)
- Safety protocol for organic subjects

**Level 2: Lighting & Atmosphere (The "Mood")**
- Define environmental lighting
- Select time of day / weather conditions
- Set emotional tone through light quality
- Control color temperature and contrast
- Add atmospheric effects (fog, volumetric)

**Level 3: Camera & Motion (The "Perspective")**
- Define observer position and movement
- Select lens and framing
- Control motion quality (smooth, organic, dramatic)
- Set emotional intent through cinematography
- Specify temporal coherence for video
